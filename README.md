## Read Me — онлайн‑читалка текста голосом

**Read Me** — это современный онлайн‑сервис синтеза речи, который превращает любые текстовые материалы в качественное озвучивание.  
Приложение работает прямо в браузере, оптимизировано для настольных и мобильных устройств и разворачивается на Cloudflare Workers/Pages.

Основные возможности:

- **Озвучивание текста** на нескольких языках.
- **Подсветка текущего слова** (эффект «караоке» при чтении).
- **Пауза, продолжение и полная остановка** чтения.
- **Скачивание озвученного текста в MP3** после генерации.
- **Автоматическое определение языка текста** и подбор голоса.
- **Сохранение состояния в браузере** — текст и позиция чтения сохраняются между перезагрузками.

Приложение разработано творческим объединением **Green Point**. Наша цель — создавать полезные и доступные цифровые продукты для людей, помогая им экономить время и улучшать качество жизни.

## Как работает Read Me (для пользователей и SEO)

Ниже — детальное описание работы сервиса, ориентированное как на пользователей, так и на поисковые системы (Яндекс, Google, Bing и другие).

### 1. Ввод текста

- Пользователь вставляет или вводит текст в большую текстовую область.
- Поддерживаются длинные тексты: статьи, заметки, письма, инструкции и другие документы.
- Приложение аккуратно работает с переносами строк, абзацами и ссылками в тексте.

### 2. Автоматическое определение языка

Внутри приложения реализован модуль **language detection**, который:

- анализирует символы и слова в тексте;
- определяет, относится ли текст к одному из поддерживаемых языков (например, русский, английский, немецкий, французский, испанский и др.);
- автоматически выбирает подходящую языковую модель для синтеза речи.

Благодаря этому озвучивание получается более естественным, а голос — корректно подобран под язык исходного текста.

### 3. Выбор и настройка голоса

В разделе **настроек** (иконка шестерёнки):

- пользователь может выбрать один из доступных голосов (нейросетевые голоса для разных языков);
- настройки голоса сохраняются в локальном хранилище браузера и подставляются автоматически при следующем запуске.

### 4. Генерация озвучки

При нажатии на кнопку **«Читать»**:

- приложение подготавливает текст к синтезу;
- формирует запрос к внешнему сервису текст‑в‑речь (TTS);
- получает готовый аудио‑поток в формате MP3;
- подготавливает данные для последующего скачивания файла.

Важно: с точки зрения пользователя процесс выглядит как обычное нажатие на кнопку и короткая загрузка, после чего начинается воспроизведение.

### 5. Воспроизведение с «караоке‑эффектом»

Во время воспроизведения:

- текст подсвечивается синхронно с ходом озвучивания,
- текущее слово выделяется более ярко,
- уже прочитанный фрагмент отображается отдельным цветом,
- непрочитанный текст отображается приглушённым цветом.

Такой режим упрощает восприятие для людей, которые одновременно **слушают и читают**: это полезно при изучении языков, работе с большими текстами и подготовке материалов.

### 6. Пауза, продолжение и остановка

Управление воспроизведением:

- **Pause / Пауза** — временно останавливает воспроизведение, при этом текущая позиция в тексте сохраняется.
- **Resume / Продолжить** — продолжает воспроизведение с того же места.
- **Stop / Сбросить** — полностью останавливает чтение и возвращает индикатор позиции в начало текста.

Позиция чтения (индекс символа) сохраняется в локальном хранилище браузера. Поэтому даже при перезагрузке страницы пользователь может:

- продолжить с того же места, где остановился (Resume),
- или принудительно сбросить состояние кнопкой Stop.

### 7. Скачивание аудио (MP3)

После успешной генерации озвучки приложение:

- сохраняет ссылку на последний сгенерированный аудиофайл;
- показывает отдельную кнопку **«Скачать MP3»**;
- позволяет загрузить файл и использовать его для личных задач: слушать офлайн, вставлять в презентации и т.д.

Если пользователь меняет текст хотя бы на один символ, ссылка на скачивание сбрасывается, чтобы избежать несоответствия между текстом и аудио.

### 8. Реклама и поддержка сервиса

Чтобы приложение оставалось бесплатным для пользователей:

- перед началом чтения и перед скачиванием MP3 показывается **небольшое рекламное модальное окно**;
- в нём есть таймер обратного отсчёта;
- по завершении таймера пользователь может продолжить чтение или скачать файл.

Реклама не вмешивается в основной интерфейс и не влияет на содержимое текстов.

### 9. Технологический стек

Приложение построено на:

- **React + TypeScript** — интерфейс и логика взаимодействия;
- **Vite** — сборка фронтенда;
- **Cloudflare Workers / Pages** — хостинг и быстрая доставка контента по всему миру.

Приложение не требует от пользователя установки программ или расширений, работает прямо в браузере.

## Запуск локально

**Требования:** установленный Node.js (актуальной версии LTS).

1. Установка зависимостей:

   ```bash
   npm install
   ```

2. Запуск в режиме разработки:

   ```bash
   npm run dev
   ```

3. Открыть в браузере:

   ```text
   http://localhost:3000
   ```

Фронтенд будет работать с публичным TTS‑сервисом, дополнительный сервер поднимать не требуется.

## Деплой на Cloudflare Workers / Pages

Проект можно подключить к GitHub‑репозиторию и собрать на Cloudflare:

- **Build command**: `npm install && npm run build`
- **Output directory**: `dist`

После сборки статический фронтенд разворачивается на домене Cloudflare. Приложение использует внешние API для озвучки, поэтому отдельный backend‑сервер не нужен.

## Политика конфиденциальности

Данный раздел описывает, как сервис **Read Me** обрабатывает данные пользователей.

- Мы **не создаём аккаунтов** и не просим регистрироваться.
- Вводимый текст используется **только для генерации озвучки** во время вашей сессии.
- Текст может временно сохраняться в **локальном хранилище браузера (localStorage)**, чтобы:
  - восстанавливать черновики текста при перезагрузке страницы;
  - запоминать выбранный язык интерфейса и голос.
- Эти данные **не отправляются на наш собственный сервер** и хранятся только на стороне пользователя.
- Для синтеза речи часть текста передаётся во внешний сервис TTS. Он используется исключительно для преобразования текста в звук.
- На странице может использоваться рекламный код (например, рекламные блоки), которые могут собирать обезличенную статистику показов и кликов в соответствии с политиками соответствующих рекламных сетей.

Если вы не хотите, чтобы текст сохранялся локально:

- очистите данные сайта в настройках браузера;
- используйте режим «инкогнито» / приватный режим.

Мы не продаём и не передаём тексты пользователей третьим лицам, за исключением технически необходимой передачи во внешний сервис синтеза речи.

Используя сервис **Read Me**, вы соглашаетесь со следующими условиями:

Мы можем обновлять функциональность сервиса и текущий документ без предварительного уведомления пользователей.
